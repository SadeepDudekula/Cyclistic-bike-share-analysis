# Cyclistic Bike Share Analysis

## Project Overview
Analysis of 4.1 million bike rides to understand 
differences between casual riders and annual members.

## Tools Used
- Python — Combined 12 months CSV files
- SQL Server (SSMS) — Data cleaning & analysis
- Tableau Public — Interactive dashboard

## Key Findings
- Casual riders peak on weekends (leisure)
- Members ride consistently on weekdays (commuting)
- Both groups peak in summer (May-August)
- Casual riders take longer rides (19.8 min vs 11.8 min)
- Members = 64% of total rides, Casual = 36%

## Recommendations
1. Weekend-only membership at lower price
2. Summer 3-month membership campaign launching April
3. Targeted digital ads at parks and cycling routes
